# Password Generator App

This is a simple Java application that generates random passwords based on user input. The application allows users to specify the length of the password and ensures that the generated password contains a mix of lowercase letters, uppercase letters, digits, and punctuation characters.

## Features

- Generate random passwords
- Ensure passwords contain a mix of character types
- User-friendly console interface

## Requirements

- Java Development Kit (JDK) 8 or higher

## Compile and Run

To compile and run the Password Generator App, follow these steps:

1. Open a terminal or command prompt.
2. Navigate to the project directory.

### Compile

```sh
javac -d bin App.java
java -cp bin App
